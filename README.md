## 🔥 Overview

A modern, user-friendly, and beautifully designed web tool for generating activation keys for **WinRAR**—the ultimate archive manager.

All processes run **100% locally** in your browser, ensuring maximum privacy and security.

## ✨ Key Features

- **💎 Premium Glassmorphism UI** - Enjoy a sleek, modern, and completely responsive experience.
- **🌓 Adaptive Theme** - Seamless dark and light mode toggle.
- **🌍 Multi-Language Support** - Effortlessly switch between English and Spanish interfaces natively.
- **💾 Auto-Download** - Get your `rarreg.key` file instantly and securely delivered to your machine.
- **♾️ Unrestricted Licensing** - Generate custom license types and usernames with no limitations.

## 🚀 Compatibility

- Official support tested extensively across **all major WinRAR versions** (v5.X, v6.X, and v7.X+).
- Compatible with all Desktop and Installer editions.

## 📖 How to Use

### 1️⃣ The Key Generator

1. **Access the Generator**: Open the [WinRAR Key Generator](https://winrar-keygen.vercel.app/) web application.
2. **Setup your License**:
   - Enter your desired **Username**.
   - Input your custom **License Type**.
3. **Generate**: Click the big **"Generate License"** button to automatically download your `rarreg.key` file.
4. **Deploy**: Drop this generated file inside your WinRAR installation folder:
   - **64-bit Systems**: `C:\Program Files\WinRAR`
   - **32-bit Systems**: `C:\Program Files (x86)\WinRAR`

*(If Windows asks for administrator permissions to paste the file, click **Yes**).*

## 🛠 Troubleshooting

<details>
<summary><strong>Activation isn't being recognized?</strong></summary>
Ensure that you paste the file directly inside the core installation folder where `WinRAR.exe` is located. Also, make sure the file is strictly named `rarreg.key` without any duplicates like `rarreg (1).key`.
</details>

## 🔐 Technical & Security Details

This tool operates completely **Client-Side** leveraging Vanilla Javascript, Alpine.js, and pure DOM manipulations. **Not a single byte of your input data or generated licenses travels to any backend server.**
